# bench-rest-test-be

Code submission for Bench rest test backend

## Install Prerequisites

- Clojure: https://clojure.org/guides/getting_started
- Leiningen: https://leiningen.org/

## Running

    $ lein run

## Tests

    $ lein tests

## Assumptions

Assumed that a maximum of 10 transactions per page.
